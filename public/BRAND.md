# Fixturday — Brand Identity

## Logo faili
- `logo-primary.svg` — 400x400px, tumšs fons, kvadrāts (OG attēls, social media)
- `logo-horizontal.svg` — 600x80px, transparents fons (website header, admin nav)
- `favicon.svg` — 512x512px, amber fons (favicon, app icon)

## Krāsas
```
--color-navy:    #0d1b2e   (primārais fons)
--color-amber:   #f0a500   (akcents, logo, pogas)
--color-surface: #16213e   (kartes, sidebar)
--color-muted:   #8892a4   (sekundārais teksts)
--color-text:    #e0e0e0   (primārais teksts)
--color-success: #2ecc71   (aktīvs, apstiprināts)
--color-info:    #3498db   (pabeigts, informatīvs)
--color-danger:  #e74c3c   (dzēst, kļūda)
```

## Tipografija
```
Barlow Condensed — virsraksti, logo, skaitļi (Google Fonts)
Inter            — pamatteksts, formas, tabulas (Google Fonts)

H1: Barlow Condensed 700, 32-40px, letter-spacing: 1-2px
H2: Barlow Condensed 700, 24-28px
H3: Barlow Condensed 600, 18-20px
Body: Inter 400, 15-16px, line-height: 1.6
Muted: Inter 400, 13px, color: --color-muted
Numbers/scores: Barlow Condensed 700, tabular-nums
```

## Ikonas

## Logo izmantošana
✅ Logo uz tumša fona — amber krāsā
✅ Logo uz gaiša fona — navy krāsā  
✅ Favicon — amber fons ar navy simbolu
✅ Horizontāls logo nav — header navigācijā
❌ Nekad nemainīt proporcijas
❌ Nekad izmantot citas krāsas
❌ Minimālais izmērs: 32px augstums

## Identitātes koncepcija
Grafika/schedule motīvs — rindas kas kļūst garākas
simbolizē spēļu grafiku, kas aizpildās ar informāciju.
Amber krāsa = enerģija, sports, uzvara.
Navy = uzticamība, profesionalitāte, vakara spēles.
